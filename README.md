# Globetrotter

- A travel activity manager, to help traveler keep track on their plannings.

## Prerequisites

- Node installed on local machine (Version 16.18.1 used during development)
- Link: https://nodejs.org/en/
- Angular CLI installed
- run "npm install -g @angular/cli"

## Development

- run "npm install" to install dependencies
- run "npm run server", provides data from local db.json file

...
